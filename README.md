# Jonathan Savastano - Portfolio

This is my portfolio website showcasing all of my game development and web development projects.

## Tech Stack

- **React** - Modern UI framework
- **Vite** - Fast build tool
- **React Router** - Client-side routing
- **CSS3** - Modern styling with glassmorphism effects

## Features

- Sleek modern design with gradient backgrounds and glassmorphism
- Fully responsive layout
- Interactive calculator component
- Digital clock
- Project showcases with videos and images
- Links to external projects (WPM counter, Weather app)

## Development

The React source code is in the `react-portfolio/` directory.

To make changes:

```bash
cd react-portfolio
npm install
npm run dev  # Start development server
npm run build  # Build for production (outputs to parent directory)
```

After building, the static files will be in the root directory, ready for GitHub Pages deployment.

## Deployment

This site is deployed on GitHub Pages and serves static files from the root directory.
